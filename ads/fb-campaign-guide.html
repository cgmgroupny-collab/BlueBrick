01

ACCOUNT SETUP CHECKLIST

PRE-LAUNCH REQUIREMENTS
  • Facebook Business Page -- Verify Blue Brick Luxury & Commercial Cleaning page is active, has logo, cover photo, contact info, and "Cleaning Service" category.
  • Meta Business Suite -- Create or access Business Suite at business.facebook.com. Add the Blue Brick page as an asset.
  • Ad Account -- Create a dedicated ad account in Business Suite > Settings > Ad Accounts. Set currency to USD, timezone to Eastern.
  • Payment Method -- Add a credit/debit card or link a bank account. Set a daily spending limit as a safety net (recommended: $100/day to start).
  • Meta Pixel -- Install the Meta Pixel on bluebrickmass.com (see Section 6 for detailed instructions).
  • Conversions API -- Set up server-side tracking via Conversions API for more accurate attribution.
  • Custom Conversions -- Create 3 custom conversion events: form_submit (lead form completed), phone_click (tel: link clicked), sms_click (sms: link clicked).
  • Domain Verification -- Verify bluebrickmass.com in Business Suite > Brand Safety > Domains.
  • Ad Creative Assets -- Upload all 10 AI-generated images to the Media Library in Ads Manager. Name them clearly (e.g., "01-Office-Bait-Switch").
  • Instagram Connection -- Connect the Blue Brick Instagram account (if applicable) for cross-platform placements.
Pro Tip: Before going live, run the Meta Pixel Helper Chrome extension on bluebrickmass.com to verify the pixel is firing correctly on page load, form submission, and button clicks.

EXCLUSIONS (CRITICAL)
  • Exclude existing customers -- Upload a customer email/phone list as a Custom Audience and exclude from all cold campaigns.
  • Exclude converters -- Exclude anyone who completed form_submit in the last 30 days from cold campaigns (they should see retargeting only).
  • Exclude competitors -- If possible, exclude job titles like "Cleaning business owner", "Janitorial services" to avoid wasting spend on competitors scouting your ads.
LOOKALIKE AUDIENCES (PHASE 2)

After collecting 100+ leads, create Lookalike Audiences:

  • 1% Lookalike from form_submit custom audience (highest quality, smallest reach)
  • 3% Lookalike from website visitors (broader reach for scaling)
  • 5% Lookalike from page engagers (broadest, use for brand awareness campaigns)

Layer the Lookalike with the geographic restriction (25mi Boston) for best results.

Boston-Specific Insight: Target "Sept 1 moving" audiences starting August 1st. Boston has the nation's largest synchronized move-in/move-out date. This is a massive seasonal opportunity for move-out cleaning ads.

Best for: Testing the waters, limited budget, proving ROI before scaling.

  • These two cover rational (comparison) and emotional (trust) angles

SCALING RULES

When to increase budget:

  • Increase by 20% when CPL is consistently below target for 5+ days
  • Never increase more than 20%/day -- larger jumps reset the learning phase
  • Kill an ad if it has spent 2x the target CPL with zero conversions
  • Pause a campaign if CPL exceeds 3x target for 7+ days
  • Add a new campaign only after existing campaigns are profitable

A/B TESTING STRATEGY

TESTING FRAMEWORK

Meta's algorithm needs enough data to find a winner. Follow these rules to avoid wasting budget on inconclusive tests.

1
TEST ONE VARIABLE
Only change one element per test: creative, headline, audience, OR placement. Never test multiple variables simultaneously.
2
WAIT 5-7 DAYS
Let each ad run for a full week before making decisions. Meta needs ~50 conversion events to exit the learning phase.
3
KILL THE LOSER
After 7 days, pause the underperforming ad. Reallocate its budget to the winner. Launch a new challenger.
TESTING PRIORITY ORDER

Test in this order for maximum ROI impact:

  1. Creative (Image) -- The biggest lever. Test the AI-generated images against each other. The scroll-stopping image determines 80% of ad performance.
  2. Primary Text -- Test short-form (under 125 chars) vs. long-form (storytelling) versions of the same ad.
  3. Headline -- Test benefit-driven ("Get Your Deposit Back") vs. curiosity-driven ("Why Contractors Keep Getting Callbacks") headlines.
  4. Audience -- Test interest-based targeting vs. Lookalike audiences once you have enough conversion data.
  5. Placement -- Test Advantage+ (automatic) vs. manual placements (Feed only, Stories only).
  6. CTA Button -- Test "Get Quote" vs. "Learn More" vs. "Send Message" to see which drives the most qualified leads.
WEEK-BY-WEEK TESTING CALENDAR

Common Mistake: Changing ads during the "Learning Phase" (first 50 conversions). Every edit resets the learning phase. Let the algorithm learn before optimizing.

META PIXEL SETUP

STEP 1: CREATE THE PIXEL
  1. Go to Meta Business Suite > Events Manager
  2. Click "Connect Data Sources" > Web > Meta Pixel
  3. Name it: "Blue Brick Website Pixel"
  4. Enter website URL: bluebrickmass.com
  5. Copy the Pixel ID (you'll need this for installation)
STEP 2: INSTALL BASE CODE

Add this code to the <head> section of every page on bluebrickmass.com (index.html, cities.html, all blog posts):

<!-- Meta Pixel Code --> <script> !function(f,b,e,v,n,t,s) {if(f.fbq)return;n=f.fbq=function(){n.callMethod? n.callMethod.apply(n,arguments):n.queue.push(arguments)}; if(!f._fbq)f._fbq=n;n.push=n;n.loaded=!0;n.version='2.0'; n.queue=[];t=b.createElement(e);t.async=!0; t.src=v;s=b.getElementsByTagName(e)[0]; s.parentNode.insertBefore(t,s)}(window, document,'script', 'https://connect.facebook.net/en_US/fbevents.js'); fbq('init', 'YOUR_PIXEL_ID_HERE'); fbq('track', 'PageView'); </script> <noscript><img height="1" width="1" style="display:none" src="https://www.facebook.com/tr?id=YOUR_PIXEL_ID_HERE&ev=PageView&noscript=1" /></noscript> <!-- End Meta Pixel Code -->
STEP 3: TRACK CUSTOM EVENTS

Add these event triggers to capture the key conversion actions on bluebrickmass.com:

// Fire when the lead form is submitted (in the form's submit handler) fbq('track', 'Lead', { content_name: 'Contact Form Submission', content_category: 'Lead Generation' }); // Fire when someone clicks the phone number document.querySelectorAll('a[href^="tel:"]').forEach(link => { link.addEventListener('click', () => { fbq('track', 'Contact', { content_name: 'Phone Click' }); }); }); // Fire when someone clicks the SMS/text link document.querySelectorAll('a[href^="sms:"]').forEach(link => { link.addEventListener('click', () => { fbq('track', 'Contact', { content_name: 'SMS Click' }); }); });
STEP 4: VERIFY INSTALLATION
  • Install Meta Pixel Helper Chrome extension
  • Visit bluebrickmass.com -- Pixel Helper should show a green "PageView" event
  • Submit a test form -- verify "Lead" event fires in Events Manager
  • Click the phone number -- verify "Contact" event fires
  • Check Events Manager > Test Events tab for real-time event stream
  • Run the Diagnostics tab in Events Manager to catch any warnings
Conversions API (CAPI): For maximum tracking accuracy (especially with iOS 14.5+ privacy changes), also set up server-side Conversions API. If using a form service like Web3Forms, check if they offer a native CAPI integration -- this is the easiest path.

RETARGETING SETUP

WHEN TO LAUNCH RETARGETING

Wait at least 14 days after cold campaigns launch. You need enough pixel data to build meaningful retargeting audiences. Minimum: 500 website visitors or 1,000 video viewers.

CUSTOM AUDIENCES TO CREATE
Audience NameSourceWindowPriority
Website Visitors Meta Pixel -- All website visitors Last 30 days High
Form Abandoners Pixel -- Visited site but did NOT trigger Lead event Last 14 days High
Video Viewers 75%+ Video engagement -- watched 75%+ of any video ad Last 14 days Medium
Page Engagers Anyone who liked, commented, shared, or clicked a Blue Brick post/ad Last 30 days Medium
Instagram Engagers Anyone who interacted with Blue Brick's Instagram profile Last 30 days Low
RETARGETING AD STRATEGY

Retargeting audiences are warm -- they already know Blue Brick. Use trust + offer creative:

  • Ad 4 (Manifesto Letter) -- Builds trust and authority. "We're not the cheapest" position resonates with people who've already shown interest.
  • Ad 10 (Free Walkthrough Offer) -- Direct-response CTA. Low-friction offer for people who are considering but haven't committed.

Rotate these two ads. Don't run both simultaneously in the same ad set. Let Meta auto-optimize between them.

EXCLUSIONS FOR RETARGETING
  • Exclude existing customers (uploaded customer list)
  • Exclude converters (people who already submitted the form in the last 30 days)
  • This ensures retargeting budget is only spent on people who visited but did NOT convert
Advanced Move: Create a "Hot Leads" retargeting audience of people who visited the site 3+ times in the last 7 days. These are your highest-intent prospects. Even $3-5/day on this micro-audience can generate the best CPL of any campaign.

WEEKLY REPORTING TEMPLATE

Every Monday, pull these numbers from Ads Manager and track in a spreadsheet:

  1. Total spend (by campaign)
  2. Total leads (form submissions + phone clicks)
  3. CPL (by campaign and by individual ad)
  4. CTR (by ad -- identifies creative fatigue)
  5. Frequency (by ad set -- identifies audience saturation)
  6. Walkthroughs booked (manual input from lead tracking)
  7. Revenue from ad leads (manual input, lag 2-4 weeks)
  8. Notes (what changed, what was paused, what was launched)
RED FLAGS -- WHEN TO ACT
SignalMeaningAction
CPL > $25 Targeting or creative isn't resonating Refresh creative first. If CPL stays high after new creative, narrow audience.
CTR < 0.8% Ad isn't stopping the scroll Replace the image. The visual is the #1 factor in CTR.
Frequency > 4 Audience is oversaturated Expand audience, add new interests, or launch a Lookalike.
High clicks, low conversions Landing page problem Check page speed (under 3 sec). Test form simplicity. Ensure mobile-friendly.
Good leads, no bookings Follow-up problem Review response time (target: under 5 min). Script follow-up texts.
Lead Response Time: Studies show leads contacted within 5 minutes are 21x more likely to convert than leads contacted after 30 minutes. Use the Blue Brick SMS auto-responder to instantly acknowledge every lead that comes in. Speed is everything.
